# ADR-0005: 個人情報を LLM に渡さない（PII/AI 境界）

- ステータス: 採用
- 日付: 2026-05-31

## コンテキスト
LLM を施策要約・文面生成に使うが、顧客の個人情報を外部 LLM に渡すことはプライバシー・コンプライアンス上避ける。

## 決定
- LLM プロンプトには匿名化・集約済みデータのみ渡す（件数・セグメント名・統計）。
- 個別顧客の識別子・氏名・連絡先・購買明細を渡さない。入力 DTO の型レベルでこれらを持たせない。
- 生成は施策／セグメント単位（顧客ごとに呼ばない）。文面はテンプレ + 差し込み。
- AI 連携は `Infrastructure/Ai` に隔離し、Application は `IAiProposalService` 越しに使う。
- プロンプト非 PII を自動テスト（WireMock 捕捉）で担保する。

## 影響
- LLM 利用コストを施策単位に抑え、粗利を毀損しない。
- 文面の個別最適度は下がるが、差し込みで実用十分とする。
