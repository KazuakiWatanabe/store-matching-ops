# ADR-0007: 効果はホールドアウトでリフト（増分）として測る

- ステータス: 採用
- 日付: 2026-05-31

## コンテキスト
配信由来の見かけの CV には「施策がなくても来た顧客」が含まれ、過大評価になりうる。

## 決定
- 各施策（またはセグメント）で対象顧客を処置群（配信, 80〜90%）/ 対照群（非配信, 10〜20%）にランダム分割する。
- リフト = 処置群CVR − 対照群CVR を主要指標とする。増分売上・増分ROI で評価。
- 割当は `experiment_assignments` に記録し、`conversion_events` と突合する。
- 1施策では検出力が不足しやすいため、同種施策をプールし信頼区間つきで判断する。

## 影響
- 対照群分の機会損失が生じる（比率は小さめ・期間限定で管理）。
- PoC の結論が増分ベースで信頼できるものになる。
